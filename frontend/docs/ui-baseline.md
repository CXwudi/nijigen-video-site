# UI Baseline

We have two UI libraries, daisyUI and Base UI.

## When to use which

Generally DaisyUI for the themeing and CSS, Base UI as the headless UI with proper a11y and interactivity.

- Always use Base UI if there is a component available for the need.
- Always use daisyUI CSS for styling.
- If no Base UI component available, just normal HTML with plain daisyUI CSS is fine.
